# webtech-208

## Brève introduction


## Prérequis
- Node.js v22 ou supérieur
- npm (Node Package Manager)
- Git
- Navigateur web (Chrome ou Firefox)

## Instructions d’installation

1. Cloner ce dépôt :

   ```bash
   git clone https://github.com/Clement-dl/webtech-208.git
   cd webtech-208

2. Installer les dépendances :
    ```bash
    npm install

## Instructions d’utilisation

1. Lancer l’application en mode normal :
   ```bash
   npm start

2. Lancer l’application en mode développement avec Nodemon :
    ```bash
    npm run dev

3. Ouvrir un navigateur et accéder à :

- [http://localhost:8080/](http://localhost:8080/) → Page d’accueil
- [http://localhost:8080/hello?name=John](http://localhost:8080/hello?name=John) → Affiche *Hello John*
- [http://localhost:8080/about](http://localhost:8080/about) → Affiche le contenu du fichier JSON
- Tout autre chemin renvoie une **erreur 404**
